# IT Documentation Generator

A Flask-based SaaS project for generating IT documentation for small businesses, MSPs, and IT consultants.

## Project Goal

The goal of this project is to build a simple, useful documentation generator that can eventually become a recurring revenue SaaS product.

## Target Users

- Small businesses
- Managed service providers
- IT consultants
- Internal IT teams

## Planned Tech Stack

- Python
- Flask
- SQLite
- HTML/CSS/JavaScript
- GitHub

## Current Status

Project setup and planning phase.
