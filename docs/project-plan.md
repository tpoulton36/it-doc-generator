# MVP User Workflow

## Goal

Allow IT professionals to convert unstructured infrastructure notes into organized documentation.

---

## User Input Example

Server: DC01
IP: 192.168.1.10
OS: Windows Server 2022

File Server
IP: 192.168.1.20

Switch
Cisco 2960
IP: 192.168.1.2

---

## User Actions

1. Open the application.
2. Paste raw notes into a text area.
3. Click Generate Documentation.
4. Review generated output.
5. Copy or export documentation.

---
